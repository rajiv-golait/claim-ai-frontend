# Claim AI Frontend

A modern React frontend for the Policy Intelligence API, providing an intuitive interface for document upload and policy querying.

## Features

- 🔐 Secure API key management (client-side only)
- 📄 Multi-file document upload (PDF/DOCX)
- 🔍 Natural language policy queries
- 💚 Real-time API health monitoring
- 📊 Beautiful results display with clause references
- 📱 Fully responsive design

## Tech Stack

- React 18 with Vite
- Material-UI (MUI) for components
- Axios for API requests
- Modern ES6+ JavaScript
